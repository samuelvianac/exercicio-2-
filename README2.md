# Exercício 02 - Experimentos com Padrões

## Descrição
Projeto Godot para experimentar com diferentes padrões de textura e configurações de tiles.

## Controles
- **Tecla 1**: Aplica textura de listras
- **Tecla 2**: Aplica textura de pontos  
- **Tecla 3**: Próxima configuração de tiles
- **Tecla 4**: Configuração anterior de tiles

## Configurações de Tiles
- 1x1, 2x2, 4x4, 8x8, 2x4, 4x2

## Como usar
1. Execute o projeto no Godot
2. Use as teclas 1 e 2 para alternar entre texturas
3. Use as teclas 3 e 4 para testar diferentes configurações de tiles

## Observação
Este projeto espera duas imagens na pasta raiz:
- `listras.png` - Textura de listras
- `pontos.png` - Textura de pontos
